# Tests

Dieses Verzeichnis enthält alle Test- und Beispieldateien.

## Test-Dateien

- `test_programming.py` - Grundlegende Programmier-Tests
- `test_full_programming.py` - Erweiterte Programmier-Tests
- `test_kreisflaeche.py` - Test des Kreisflächenprogramms
- `test_kreisflaeche_debug.py` - Debug-Version des Kreisflächentests
- `test_sto_rcl.py` - Tests für Speicher-Funktionen
- `test_gui_sto_rcl.py` - GUI-Tests für STO/RCL
- `test_sst.py` - Test des Single-Step Trace
- `test_load_program.py` - Test zum Laden von Programmen
- `debug_rs.py` - Debug-Hilfe für R/S Funktionalität

## Beispiele

- `examples.py` - Verschiedene Beispiele für die Verwendung des Rechners

## Test-Ausführung

```bash
cd ..  # Zurück zum Hauptverzeichnis
python tests/test_programming.py
python tests/test_kreisflaeche.py
```
